Thank you for your interest in Macromedia Dreamweaver UltraDev 4.0

In order to install and run this sample application, you must:

1. Make sure that the ASP files and all of the dependent assets (graphics files, .htm files, etc.)
   are within the scope of an IIS Server. (\Inetpub\wwwroot\yoursitename)

2. Create an ODBC System DSN that points to the included Access Database.  For more detailed information regarding this
   process, please refer to the portion of the Online Documentation entitled, "Setting Up a DSN in Windows". 
   (Appendix E, page 249 of the "Using UltraDev" Documentation)
